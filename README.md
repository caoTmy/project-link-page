# project-link-page

**Live:** `http://tmy.dev-craft.de/`

My personal link page inspired by the looks of business cards — a central place for my social profiles and a quick tech stack overview.

Built with simple HTML & CSS. No dependencies, no build step.

## Customize

- **Social links** — update the `href` attributes in the `.links` section of `index.html`
- **Tech stack** — add or remove `<li class="tech-item">` entries under Languages, Frameworks, and Deployment
